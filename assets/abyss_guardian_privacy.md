## 1. Introdução

Esta Política de Privacidade explica como o bot **Abyss Guardian** coleta e utiliza dados necessários para seu funcionamento dentro da plataforma Discord.

---

## 2. Dados Coletados

O bot pode coletar apenas dados técnicos necessários para funcionamento:

- ID de usuário do Discord
- ID de servidor
- ID de canal
- comandos executados
- configurações do servidor

Nenhum dado sensível é coletado intencionalmente.

---

## 3. Finalidade do Uso dos Dados

Os dados coletados são utilizados para:

- funcionamento dos comandos
- armazenamento de configurações
- prevenção de abuso
- diagnóstico de erros

---

## 4. Compartilhamento de Dados

O bot não vende nem compartilha dados pessoais.

Alguns dados podem ser processados por serviços técnicos necessários ao funcionamento do bot, como hospedagem ou banco de dados.

---

## 5. Segurança

Medidas técnicas são utilizadas para proteger os dados armazenados contra acesso não autorizado.

Apesar disso, nenhum sistema online pode garantir segurança absoluta.

---

## 6. Retenção de Dados

Os dados são armazenados apenas pelo tempo necessário para funcionamento do bot.

Quando não forem mais necessários, podem ser removidos.

---

## 7. Direitos dos Usuários

Usuários podem solicitar:

- informações sobre dados armazenados
- correção de dados
- remoção de dados quando aplicável

Solicitações podem ser feitas através do email informado.

---

## 8. Alterações nesta Política

Esta política pode ser atualizada para refletir mudanças legais ou técnicas.
